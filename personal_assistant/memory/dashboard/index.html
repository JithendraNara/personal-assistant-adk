Add to ~/.claude/settings.json or Claude Desktop config: 

{
  "mcpServers": {
    "unified-memory": {
      "command": "python3",
      "args": ["-m", "personal_assistant.memory.server"]
    }
  }
}

🌐 REST API (OpenClaw, Codex, Custom)

Start the REST server and use HTTP endpoints: 

# Start the memory API server
python3 -m personal_assistant.memory.server --transport rest

# Endpoints:
POST   /memories           — Add memories
POST   /memories/search    — Search memories  
GET    /memories/profile/  — Get user profile
GET    /memories/stats     — Statistics
DELETE /memories/{id}      — Forget a memory

🐍 Python SDK

from personal_assistant.memory.engine import UnifiedMemoryEngine

engine = UnifiedMemoryEngine()

# Add a memory
await engine.add("User prefers dark mode", container_tag="user_123")

# Search
results = await engine.search("preferences", container_tag="user_123")

# Get profile
profile = await engine.profile("user_123")
print(profile.static_facts)   # Long-term facts
print(profile.dynamic_context) # Recent context

📄 OpenAPI Docs

POST /memories

Add new memories with automatic fact extraction.

{
  "content": "I'm a senior engineer working on ADK",
  "container_tag": "jeethendra",
  "source": "manual",
  "auto_extract": true
}

POST /memories/search

Hybrid semantic + keyword search.

{
  "query": "What programming languages does the user know?",
  "container_tag": "jeethendra",
  "limit": 5,
  "mode": "hybrid"
}

GET /memories/profile/{tag}

Auto-built user profile with static facts and dynamic context.

// Response
{
  "container_tag": "jeethendra",
  "static_facts": ["Senior engineer", "Uses Python", "Prefers dark mode"],
  "dynamic_context": ["Building memory system", "Working on ADK project"]
}
